# Simple Books Manager

Super simple book CRUD. React + Vite frontend, C++ (Crow) backend.

## Run it

Backend (scripts in `server/`):

```bash
# Use Git Bash on Windows, or any bash on macOS/Linux
cd server
./b.sh   # build
./r.sh   # run (serves on http://localhost:3001)
# optional
./rb.sh  # clean rebuild
```

Frontend (`client/`):

```powershell
cd client
npm install
npm run dev
```
